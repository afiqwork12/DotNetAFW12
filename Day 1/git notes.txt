to get git version installed in the system

git --version

to create a local repo

git init

to check the status of the local repo

git status

to add a file to the local repo for tracking

git add <filename>

to add all files in current folder for tracking

git add -A

to commit the file to the local repo

git commit -m "commit msg(e.g. work you've done)"

to set the identity for local repo

git config --global user.email "email@email.com"

link local repo with remote repo

git remote add origin https://github.com/afiqwork12/dotnet17012022afw12.git

to check the remote repo connection 

git remote --v

to pull from a remote repo all new

create a local folder and in the cmd prompt
get clone <remote repo>
